# 🚀 Kartikey Pandey - Premium Developer Portfolio

Welcome to the repository of my personal developer portfolio. This project is a highly interactive, premium web experience showcasing my creative development skills, fluid animations, and modern frontend architecture.

🔗 **[View the Live Website Here](https://portfolio-kartikey-woad.vercel.app/)** *(Update link if hosted elsewhere)*

## ✨ Key Features

- **Fluid Animations & Micro-Interactions:** Smooth GSAP-powered reveals and custom timeline animations on load.
- **Advanced Scroll Physics:** Seamless custom smooth scrolling implemented via Locomotive Scroll.
- **Premium Aesthetics:** High-end design tokens including dynamic color-inversion (mix-blend-mode), tactile film grain (noise overlays), and infinite marquee tickers.
- **Interactive 3D Elements:** Parallax 3D tilt effects responding dynamically to cursor position using pure JavaScript.
- **Live Contact Integration:** A fully functional backend-less contact form powered by Web3Forms for instant email delivery.

## 🛠️ Technology Stack

- **HTML5 & CSS3:** Semantic markup and advanced CSS grid/flexbox layouts. Custom typography (Gilroy, Mazius) and dynamic `vw` based responsive scaling.
- **JavaScript (Vanilla):** DOM manipulation, custom event listeners, and interaction logic.
- **GSAP (GreenSock):** Industry-standard library used for robust timeline animations and element reveals.
- **Locomotive Scroll:** Virtual scrolling library used for parallax effects and buttery-smooth page navigation.
- **Web3Forms:** Serverless form submission handling.

## 🚀 Running Locally

If you'd like to run this project locally to explore the code:

1. Clone this repository:
   ```bash
   git clone https://github.com/Kartikeyp3202/Portfolio-Kartikey.git
   ```
2. Navigate into the directory:
   ```bash
   cd Portfolio-Kartikey
   ```
3. Open `index.html` in your browser, or for the best experience, use a local server like VS Code's "Live Server" extension to ensure all assets and scripts load correctly without CORS issues.
